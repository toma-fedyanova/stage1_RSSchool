Lodash

Lodash is a JavaScript library that helps programmers write more concise and maintainable JavaScript.

With the release of ES6 and later versions of JavaScript, many methods have appeared that extend the functionality of the language. For example, there are new methods of arrays and strings, as well as useful operators such as spread and rest. However, utility libraries like Lodash continue to be relevant, as they contain many useful methods that are still not available in JavaScript out of the box.

Documentation for Lodash

https://lodash.com/?_blank.

Lodash can be broken down into several main areas

Object: accessing, extending, merging, defaults, and transforming Array: creating, splitting, combining, modifying, and compressing String: conversion functions for performing basic string operations, such as trimming, converting to uppercase, camel case, etc Collection, Seq, Function

https://lodash.com/docs/4.17.15

Disadvantages of Lodash:

Lodash methods take more time to execute than normal JavaScript functions. In JavaScript, we can chain many functions together as we want but in Lodash we can’t chain functions, we can only wrap them.g

In Node.js

“Array” Methods

Lodash _.chunk() Method.

Parameters: This function accepts two parameters(array, size)

It returns the array of chunks that is also an array

“Array” Methods

Lodash _.compact() Function.

Parameters: This function accepts only a single parameter array

This function returns the array after filtering the values. The values false, null, 0, “”, undefined, and NaN are falsey.

“Array” Methods

Lodash _.difference() Function.

Parameters: This function accepts two parameters(array, values). We can use single value or the array of values

It is the array of values that are to be removed from the original array.

“Object” Methods

Lodash _.assign() Method.

The source objects are applied from left to right and any subsequent sources overwrite the property assignments of the previous sources. _.assign( dest_object, src_obj )

This method accepts two parameters: the destination object, the source objects.

“Array” Methods

Lodash _.defaults() Method.

This method accepts two parameters: the destination object, the source objects. _.defaults( dest_object, [src_obj])

This method returns an object. This method mutates object.
